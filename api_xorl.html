Logical xor

Description

Returns the logical xor of the arguments. Defined over \(ùîπ √ó ùîπ\).

C base API (generic)

#define vxorl(a0, a1, type)
#define vxorl_e(a0, a1, type, simd_ext)

C++ base API (generic)

template <typename A0, typename A1, typename T> typename simd_traits<T, NSIMD_SIMD>::simd_vectorl xorl(A0 a0, A1 a1, T);

C++ advanced API

template <typename T, typename SimdExt> packl<T, 1, SimdExt> xorl(packl<T, 1, SimdExt> const& a0, packl<T, 1, SimdExt> const& a1);
template <typename T, int N, typename SimdExt> packl<T, N, SimdExt> xorl(packl<T, N, SimdExt> const& a0, packl<T, N, SimdExt> const& a1);
