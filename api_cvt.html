Convert vector

Description

Convert input vector into a different vector type. The output type must have same length as input type.

C base API (generic)

#define vcvt(a0, from_type, to_type)
#define vcvt_e(a0, from_type, to_type, simd_ext)

C++ base API (generic)

template <typename A0, typename F, typename T> typename simd_traits<T, NSIMD_SIMD>::simd_vector cvt(A0 a0, F, T);

C++ advanced API

template <typename ToPackType, typename T, typename SimdExt> ToPackType cvt(ToPackType, pack<T, 1, SimdExt> const& a0);
template <typename ToPackType, typename T, int N, typename SimdExt> ToPackType cvt(ToPackType, pack<T, N, SimdExt> const& a0);
template <typename ToPackType, typename T, int N, typename SimdExt> ToPackType cvt(pack<T, N, SimdExt> const& a0);
